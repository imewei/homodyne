The user guide is organized to take you from installation through your first analysis and beyond:

1. **Installation**: Platform-specific installation instructions (Linux, macOS, Windows)
2. **Quickstart**: Your first analysis in 5 minutes
3. **Configuration**: Understanding the YAML configuration system
4. **CLI Usage**: Command-line interface reference
5. **Shell Completion**: Setting up bash/zsh autocompletion
6. **Examples**: Real-world usage examples for common workflows
